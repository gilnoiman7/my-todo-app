# מנהל המשימות שלי - פרויקט סיום ריאקט

הכירו את TO-DO - פרויקט הגמר שלי בקורס React, שנולד מתוך שאיפה לשלב פשטות מקסימלית עם פונקציונליות יומיומית. האפליקציה מאפשרת ניהול משימות אינטואיטיבי תוך שימוש ב-Local Storage, מה שמבטיח שהמידע שלכם נשמר מקומית וזמין עבורכם בכל רגע. כלי פשוט שעושה סדר ביום יום ועוזר לכם להתמקד במה שחשוב.

## מה האפליקציה יודעת לעשות?
* להוסיף משימות חדשות (כמובן).
* לסמן ב-V משימות שסיימתי או להחזיר אותן למצב פעיל.
* לערוך טקסט של משימה אם טעיתי בכתיבה.
* למחוק משימות שכבר לא רלוונטיות.
* לסנן את הרשימה כדי לראות רק את מה שסיימתי או רק את מה שנשאר לי לעשות.
* לשמור הכל - גם אם אני מרענן את הדף, המשימות נשארות כי הן נשמרות בזיכרון של הדפדפן (LocalStorage).

## איך מפעילים את זה ?
1. קודם כל מתקינים את הספריות: `npm install`
2. מפעילים את הפרויקט: `npm run dev`
3. נכנסים לכתובת שהטרמינל נותן

## הקומפוננטות ואחריותן
* App: זה הקובץ הראשי שמנהל את כל רשימת המשימות ואת הלוגיקה של השמירה
* TaskInput: החלק שאחראי על התיבה שבה כותבים משימה חדשה
* TaskFilters: הכפתורים למטה שמאפשרים לי לבחור אם לראות הכל, רק מה שפעיל או רק מה שגמור
* TaskList: הרכיב שמקבל את רשימת המשימות ומציג אותן אחת מתחת לשנייה
* TaskItem: זה העיצוב של שורה אחת של משימה, כולל כפתורי המחיקה והעריכה

## מגבלות
* המשימות נשמרות רק על המחשב שבו כתבתם אותן. אם תפתחו את האתר ממחשב אחר, הרשימה תהיה ריקה.
* העיצוב נראה הכי טוב על מסך מחשב רגיל, הוא פחות מותאם לטלפונים כרגע.
